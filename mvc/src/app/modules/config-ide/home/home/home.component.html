2.1 Elección de un lenguaje y framework adecuado para MVC

La elección del lenguaje de programación y el framework es un paso crucial en el desarrollo de sistemas de información basados en el patrón de diseño Modelo-Vista-Controlador (MVC). Algunos aspectos a considerar incluyen

Lenguaje de Programación:
Debes seleccionar un lenguaje que sea adecuado para el desarrollo web y sea compatible con el patrón MVC. Algunos ejemplos populares incluyen:
JavaScript/TypeScript: Utilizado con frameworks como Angular, React o Vue.js.
Python: Utilizado con Django o Flask.
PHP: Utilizado con Laravel, Symfony o CodeIgniter.
Java: Utilizado con Spring Framework.

Framework MVC: Cada lenguaje tiene su propio conjunto de frameworks MVC. La elección del framework dependerá de factores como la complejidad del proyecto, la comunidad de desarrolladores, la escalabilidad y las características requeridas.

Requisitos del Proyecto: Debes considerar los requisitos específicos de tu proyecto, como la funcionalidad, la escalabilidad, la seguridad y la facilidad de mantenimiento. Esto te ayudará a determinar qué lenguaje y framework se ajustan mejor a tus necesidades.

2.2 ¿Que lenguaje de programación o framework vamos a manejar?

A la hora de desarrollar nuestra tienda virtual "Aprendiendo a desarrollar sistemas de información basados en MVC", hemos tomado decisiones cuidadosas sobre las tecnologías a utilizar. Si bien existen numerosos lenguajes y frameworks para el desarrollo web, hemos optado por Python con Flask como nuestro lenguaje principal y marco de trabajo. A continuación, te explicamos por qué hemos tomado esta decisión:

  • Python y Flask: Python es conocido por su legibilidad y simplicidad, lo que lo hace ideal para enseñar a los principiantes. Flask, un microframework de Python, proporciona las herramientas esenciales para desarrollar aplicaciones web de manera eficiente. Es fácil de aprender y permite una rápida implementación de proyectos, lo que lo hace perfecto para nuestro software educativo.

  • Jinja2: Para la capa de presentación de nuestra tienda virtual, utilizaremos Jinja2, un motor de plantillas que se integra perfectamente con Flask. Jinja2 simplifica la creación de vistas dinámicas al permitir la inserción de datos dinámicos en las páginas web. Los estudiantes aprenderán a diseñar plantillas y a crear páginas web interactivas de una manera eficiente.

  • SQLAlchemy: En cuanto a la gestión de bases de datos, hemos seleccionado SQLAlchemy, una biblioteca ORM (Mapeo Objeto-Relacional) que simplifica la interacción con bases de datos. Este enfoque facilita a los estudiantes el aprendizaje sobre cómo interactuar con bases de datos relacionales y les brinda una base sólida para futuros proyectos.

En resumen, hemos elegido Python con Flask, Jinja2 y SQLAlchemy como nuestro conjunto de tecnologías para garantizar que nuestros estudiantes tengan una experiencia de aprendizaje efectiva y sólida. Estas tecnologías ofrecen un equilibrio óptimo entre simplicidad y potencia, lo que les permitirá comprender y aplicar con éxito el patrón de diseño Modelo-Vista-Controlador (MVC) en el desarrollo de sistemas de información.

2.4 Instalación del Entorno de Desarrollo

Visual Studio Code (VS Code) es un entorno de desarrollo de código abierto y altamente personalizable que se ha convertido en una herramienta esencial para desarrolladores de software en todo el mundo. Su facilidad de uso, flexibilidad y una amplia gama de extensiones lo convierten en una elección popular tanto para principiantes como para desarrolladores experimentados. A continuación, te guiaremos a través de los pasos para instalar VS Code en tu sistema.

Pasos para la Instalación

  • Descargar el Instalador Visita el sitio web oficial de Visual Studio Code en https://code.visualstudio.com/. Aquí encontrarás el enlace de descarga correspondiente a tu sistema operativo (Windows, macOS o Linux). Haz clic en el enlace para comenzar la descarga.

  • Instalar en Windows:
    Una vez que se complete la descarga, haz doble clic en el archivo de instalación.
    Se abrirá el asistente de instalación. Sigue las instrucciones en pantalla y acepta los términos de la licencia si estás de acuerdo.
    Puedes elegir opciones de instalación adicionales, como agregar VS Code al menú contextual del Explorador de Windows.
    Haz clic en "Instalar" para comenzar la instalación. Una vez que se complete, podrás iniciar Visual Studio Code.

  • instalar en macOS
    Abre el archivo descargado, que generalmente se encuentra en la carpeta "Descargas".
    Arrastra y suelta la aplicación de VS Code en la carpeta "Aplicaciones" para instalarla.
    Una vez que se complete la instalación, podrás encontrar VS Code en tu carpeta de Aplicaciones. Haz clic en el icono para iniciarlo.

  • Instalar en Linux
    La instalación en Linux puede variar según la distribución. Puedes encontrar instrucciones detalladas en la documentación oficial de VS Code para tu distribución específica.

Comenzar con Visual Studio Code

Una vez que hayas instalado VS Code, estarás listo para comenzar a escribir y depurar código. Algunas características notables de VS Code incluyen:

  • Interfaz de usuario intuitiva: La interfaz de usuario limpia y organizada de VS Code te permite enfocarte en tu trabajo sin distracciones.

  • Extensiones personalizadas: Puedes personalizar VS Code instalando extensiones para lenguajes de programación, depuración, control de versiones y más.

  • Depuración integrada: VS Code ofrece herramientas de depuración que te permiten identificar y solucionar errores en tu código de manera eficiente.

  • Integración con Git: Puedes gestionar tus repositorios de Git directamente desde VS Code.

Visual Studio Code te proporciona un entorno de desarrollo eficaz y versátil que te ayudará a desarrollar tus habilidades de programación y afrontar proyectos de desarrollo de software con confianza.

Python: Un Lenguaje Versátil

Python es un lenguaje de programación versátil y ampliamente utilizado que es conocido por su sintaxis clara y legible. Aunque este curso no se enfoca en enseñar Python desde cero, aquí tienes un resumen de los conceptos esenciales que necesitas conocer

Variables y Tipos de Datos

  • En Python, puedes declarar variables y asignarles valores.
  • Python admite varios tipos de datos, como números (enteros y flotantes), cadenas de texto y listas.

Estructuras de Control

  • Las estructuras de control, como bucles (for y while) y condicionales (if), te permiten controlar el flujo de un programa.

Funciones

  • Las funciones son bloques de código reutilizables que pueden recibir argumentos y devolver resultados.
  • En Python, puedes definir funciones utilizando la palabra clave def.

Librerías y Módulos

  • Python ofrece una amplia variedad de librerías y módulos que amplían su funcionalidad. En este curso, nos enfocaremos en Flask para el desarrollo web.

Breve Introducción a Flask, Jinja2 y SQLAlchemy

  • Flask es un microframework de Python para el desarrollo web. Aprenderemos a crear aplicaciones web utilizando Flask y comprenderemos cómo manejar rutas y vistas.
  • Jinja2 es un motor de plantillas que permite la creación de páginas web dinámicas. Aprenderemos a usar Jinja2 para integrar datos en nuestras vistas.
  • SQLAlchemy es una biblioteca ORM que facilita la interacción con bases de datos. Aprenderemos a utilizar SQLAlchemy para gestionar datos de la base de datos en nuestras aplicaciones web.

Esta breve introducción a Python tiene como objetivo proporcionar a todos los estudiantes una base sólida para comprender los conceptos clave en el desarrollo web con Flask, Jinja2 y SQLAlchemy. A medida que avanzamos en el curso, exploraremos estos conceptos en profundidad, lo que te permitirá desarrollar aplicaciones web de manera efectiva.
